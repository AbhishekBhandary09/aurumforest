# 🌿 AurumForest — Luxury Travel & Expedition
 
> *Where the World Reveals Itself*
 
A high-fidelity luxury travel landing page with an immersive 3D globe, AI concierge, Golden Hour theming, animated flight arcs, and magnetic UI.
 
## ✦ Live Demo
**https://AbhishekBhandary09.github.io/aurumforest/**
 
## ✦ Features
| Feature | Tech |
|---------|------|
| 🌍 Interactive 3D Globe | Three.js WebGL |
| ✈ Animated Flight Arcs | Bézier + quaternion |
| 🌅 Golden Hour Theming | Geolocation + CSS vars |
| 🤖 AI Travel Concierge | Claude API |
| 📜 Scroll Animations | GSAP + ScrollTrigger |
| 🧲 Magnetic Buttons | Lerp physics |
| 🖱 Custom Cursor | RAF lag ring |
| 📱 Fully Responsive | Mobile/Tablet/Desktop |
 
## ✦ Quick Start
```bash
git clone https://github.com/YOUR_USERNAME/aurumforest.git
cd aurumforest
open index.html   # or: npx serve .
```
